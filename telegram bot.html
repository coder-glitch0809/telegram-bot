AI Telegram Bot + Google Sheets xarajatlar

Asosiy kod bot.py faylida. Bu HTML sahifa faqat sozlash yo'riqnomasi.

API key joyi

Adashmaslik uchun Telegram bot tokeni va OpenAI API key bitta joyda turadi: .env.

TELEGRAM_BOT_TOKEN=PASTE_TELEGRAM_BOT_TOKEN_HERE
OPENAI_API_KEY=PASTE_OPENAI_API_KEY_HERE

Google Sheets

Service account JSON faylini shu papkaga google-service-account.json nomi bilan qo'ying.

GOOGLE_SERVICE_ACCOUNT_FILE=google-service-account.json
SHARE_SPREADSHEET_WITH_EMAIL=sizning_gmailingiz@gmail.com

Ishga tushirish

python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python bot.py

Buyruqlar

/start
/ai savol
